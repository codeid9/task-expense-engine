# CLI Task & Expense Engine

A command-line application built with TypeScript to manage expenses and tasks.

## Features

- Add expenses
- Track income and expenses
- Category-based expenses
- Type-safe data using TypeScript

## Tech Stack

- TypeScript
- Node.js
