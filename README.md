# Task & Expense Engine

A TypeScript-based task and expense manager with a graphical interface and local storage support.

## Features

- Add expenses
- Track income and expenses
- Category-based expenses
- Type-safe data using TypeScript

## Tech Stack

- TypeScript
- Node.js

## Project Structure

src/
├── models/
├── managers/
├── utils/
└── index.ts
app/
├── HTML
├── CSS
└── TypeScript
